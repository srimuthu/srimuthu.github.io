My Projects.

Display ECU Test Automation


Proposed the idea and spearheaded the project for automatic regression testing of On screen and Data-Link parameters of Display ECUs using CV and OCR, improving quality and velocity of testing procedures. Built the complete framework on Python.

Data Link Simulator for HIL simulations


Worked as a part of a research project in developing customized data link simulator with multiple protocol support (including CAN/J1939 public and proprietary protocols) for HIL simulations during development and testing of ECUs, implemented using Zero Message Queue (ZMQ) high performance asynchronous messaging library
