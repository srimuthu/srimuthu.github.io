My Projects.

Display ECU Test Automation


Worked on a project for automatic testing and validation of Display ECUs using Computer Vision and Optical Character Recognition.

Senior Year Thesis


“Condition monitoring and fault diagnosis of induction motors” - predicting impending faults in industrial induction motors by recognizing patterns in the collected sensor data transmitted by the motor to a remote processing centre

Rubik's cube solver


Built an autonomous solver robot that can mechanically solve a 3X3 Rubik's cube.The solver consists of three individual 2 DOF robotic arms controlled by RaspberryPi and the algorithm was implemented using OpenCV-Python

See some of my other projects

Self-adjusting podium mic


Designed and built a cost effective stand for podium microphone that automatically adjusts the position of the mic according to the speakers orientation. The face detection and tracking was implemented using Haar cascade filters in OpenCV on the RaspberryPi processor board

Automated Guided Vehicle


A completely autonomous, self-guided industrial robot that can continuously transport products from one machine to another in an assembly line. Our design specifically catered to the needs of the paper industry, to transport paper rolls. The project was funded by CEG Tech Forum and displayed during Kurukshetra 2011 as the K! 11 Mega Project

UGC Sponsored Project

4 DOF Robotic Arm


A fully automated robotic arm with 4 Degrees of Freedom that can draw basic shapes. Dynamic model was created using SIMULINK and the position data for the servo motors were wirelessly transmitted to the Arduino controlled robotic arm
